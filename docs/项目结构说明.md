## 🔧 技术栈

### 后端

- **Python Flask**: Web 框架
- **SQLite**: 数据库
- **Flask-CORS**: 跨域支持
- **Flask-JWT-Extended**: JWT 认证

### 前端

- **HTML5**: 页面结构
- **CSS3**: 样式和动画
- **JavaScript ES6+**: 交互逻辑
- **Fetch API**: 与后端通信

### 设计

- **响应式设计**: 支持多设备
- **现代 UI**: 渐变、阴影、动画
- **用户体验**: 流畅的交互反馈

## 📋 功能特性

### 用户系统

- 用户注册/登录
- 密码加密存储
- JWT 身份验证

### 答题模式

- **速答模式**: 60 秒限时答题
- **学习模式**: 无限时间，错题提示

### 数据库设计

- 题目随机获取
- 选项随机排列
- 答题记录存储
- 排行榜统计

### 用户体验

- 实时反馈动画
- 响应式设计
- 流畅页面切换
- 详细错题解析
